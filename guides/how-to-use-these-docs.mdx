Start with the page that matches your question. Analog's core frame is simple: it is the funding layer that makes community-led IRL safer, more measurable, and easier to repeat for Sponsors, Hosts, and attendees.

## If You Are A Company Stakeholder

Use [Company Stakeholder Brief](/guides/stakeholder-brief) first.

| Audience | Read next | What to look for |
|---|---|---|
| Employees | [Current Thesis](/strategy/current-thesis), [Lean GTM Operating Model](/strategy/lean-gtm-operating-model) | What work matters. |
| Investors | [12-Month Market Focus](/strategy/twelve-month-market-focus), [Pricing Models And Experiments](/strategy/pricing-models), [Testing Grid](/market/testing-grid) | What proves scale. |
| Sponsors | [Sponsor Personas And Pipeline](/strategy/sponsor-personas), [Proof And Spend](/platform/proof-and-spend) | How budget and proof work. |
| Hosts/connectors | [Host Models And Personas](/strategy/host-models-personas), [Host Workflow](/platform/host-workflow) | How funded work repeats. |
| Partners | [CreatorCommerce Demand Bridge](/strategy/creatorcommerce-demand-bridge), [Product Primitives](/references/product-primitives) | Where Analog embeds. |

## If You Are Planning A Pilot

1. Read [Current Thesis](/strategy/current-thesis).
2. Check focus with [12-Month Market Focus](/strategy/twelve-month-market-focus).
3. Pick a use case in [MVP Use Cases](/strategy/mvp-use-cases).
4. Separate money and proof with [Pricing Models And Experiments](/strategy/pricing-models) and [Proof And Spend](/platform/proof-and-spend).
5. Score it with [Testing Grid](/market/testing-grid).

## If You Are Running GTM

1. Start with [Lean GTM Operating Model](/strategy/lean-gtm-operating-model).
2. Turn names into briefs with [Sponsor Personas And Pipeline](/strategy/sponsor-personas).
3. Price the motion with [Pricing Models And Experiments](/strategy/pricing-models).
4. Use [Supply Motion](/market/supply-motion) and [Event Prospecting Catalog](/market/event-prospecting-catalog) to find host-side options.

## If You Are Activating Supply

1. Read [Host Models And Personas](/strategy/host-models-personas).
2. Choose access, create, publish, or prospect with [Supply Motion](/market/supply-motion).
3. Use [City Leads And Host Unlocks](/market/city-leads-host-unlocks) for local trust work.
4. Use [Asset Need Intelligence](/market/asset-need-intelligence) when blockers repeat.

## If You Are Using The Product

1. Read [Product Primitives](/references/product-primitives).
2. Read [Platform Logic](/references/platform-logic).
3. Use the how-to page for the task: [Setup Sponsor Campaign](/how-to/setup-sponsor-campaign), [Setup Host Event](/how-to/setup-host-event), [Build Event Prospecting List](/how-to/build-event-prospecting-list), [Capture Field Context](/how-to/capture-field-context), [Run Creator-Led Activation](/how-to/run-creator-led-activation), or [Location Activation](/how-to/location-activation).

## If You Are Updating These Docs

Treat new founder-provided use cases and corrections as binding. Use [Source Context Map](/references/source-context-map) to understand upstream ownership, then use [Update These Docs](/how-to/update-this-brain) to decide which pages must change.
