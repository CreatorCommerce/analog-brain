The sequence:

```text
why now
-> what Analog is
-> why Analog wins
-> where Analog focuses first
-> how markets form
-> how supply gets acquired
-> who funds and hosts
-> how product objects work
-> how operators run the motion inside Platform
```

## Reading Rule

Read top-down when you are learning the company. Jump by job when you are operating.

If a later page feels abstract, move one step back. The earlier page should define the concept the later page applies.
