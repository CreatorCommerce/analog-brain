## Start With The Shape

Analog has two motions that share the same product objects:

1. **Selective sponsor-funded pilots**: a sponsor funds a controlled IRL test, Analog structures the campaign, a host executes the activation, spend is watched, proof is collected, and the next unlock is scored.
2. **Market formation**: city leads, grants, Requests for Events, Requests for Communities, hosted forms, Play Fair, and owned activations create new sponsor-ready host supply before the marketplace is liquid.

The MVP product in `analog-discovery` currently centers on Sponsors, Hosts, Campaigns, Event Sets, Collabs, Host Events, spend cards, hosted event pages, files/docs, outreach, application portals, tasks, messages, shipments, and sponsor-visible proof.

The first clean vertical from the seed deck is running and adjacent movement communities: dense supply, recurring rituals, obvious use of funds, and clear sponsor demand.
