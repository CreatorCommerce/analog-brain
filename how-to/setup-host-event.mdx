## Inputs

Before creating the host event, confirm:

- host
- active sponsor/collab
- available allocation
- event start and end time
- approved spend categories
- event location or context
- proof requirements

## Steps

1. Open the host workspace.
2. Create a new event.
3. Select the sponsor budget.
4. Confirm the card and available amount.
5. Add timing and event details.
6. Use the spend card for approved operational spend.
7. Watch transactions during the event window.
8. Collect proof after the event.

## Proof Checklist

Collect:

- spend transactions
- receipts or vendor notes
- attendee count
- photo/video links
- sponsor-visible post or story mentions
- host notes on what worked and what blocked execution

## Renewal Notes

After the event, answer:

- Did the event create consumer or community value?
- Was sponsor money used responsibly?
- What proof would the sponsor trust?
- What would make the next activation cheaper or easier?
- Should the host unlock more budget?
