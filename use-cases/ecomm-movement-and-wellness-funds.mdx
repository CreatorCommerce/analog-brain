This page is for eComm Brands whose products become more believable inside a physical ritual: running, walking, Pilates, recovery, fertility wellness, hydration, supplements, apparel, shoes, or performance accessories.

This is a proof case, not the whole strategy. Use [Cash Funding Playbooks](/strategy/cash-funding-playbooks) for the broader doctrine: Analog should build around cash-based Host enablement across many community families, not only movement communities and not product-only gifting.

The core motion is:

```text
Sponsor budget -> movement or wellness Event set -> trusted Host -> class, club, or event -> product benefit + proof -> renew the best ritual
```

## Use Cases

| Use case | Example Sponsor shape | Best Hosts | What Analog proves |
|---|---|---|---|
| Run club performance fund | On Running-style footwear, apparel, socks, recovery, hydration, or nutrition brand | Run clubs, walk clubs, coaches, recovery studios, race crews, local route connectors | Whether recurring movement communities create product trial, attendance, content, category growth, and repeatable Host unlocks. |
| Pilates and fertility wellness series | Bird&Be-style fertility supplement or women's health brand | Pilates instructors, wellness studios, fertility educators, doulas, nutritionists, community hosts | Whether a sensitive health category can show up with permission, education, care, and useful proof rather than generic sampling. |
| Recovery morning circuit | Recovery, hydration, protein, electrolyte, skincare, or wearable brand | Saunas, cold plunge spaces, gyms, studios, wellness cafes, trainers | Whether post-workout rituals create high-intent product trial and social proof. |
| Wellness dinner or salon | Supplement, hormone health, sleep, gut health, or longevity brand | Nutritionists, doctors where appropriate, creators, private dinner hosts, studios | Whether education-led rooms create trust, opt-ins, referrals, and repeat demand. |

## On Running-Style Run Club Fund

Use this when the Sponsor wants to grow a movement category, not just place a logo on a race.

| Component | Setup |
|---|---|
| Campaign | Run Club Performance Fund. |
| Event sets | NYC run clubs, women's run crews, beginner 5K groups, race-prep crews, recovery mornings, walk/run hybrids. |
| Host budget | Coach or Host stipend, hydration, coffee, bag check, route support, photographer, recovery perk, print/signage, affordable space, product demo when useful, or post-run breakfast. |
| Product role | Shoe try-on, apparel trial, recovery item, electrolyte or nutrition sample, QR claim, or local retail tie-in. |
| Proof | RSVPs, attendance, repeat runners, product trials, claims, content, Host note, sentiment, and which clubs deserve renewal. |

Good first tests:

- sponsor a recurring neighborhood run without changing its identity
- fund a beginner-friendly training series before a local race
- pair a run with a recovery studio or coffee stop
- give top Hosts monthly allocations instead of one giant event budget

Avoid turning the club into a forced brand activation. The Sponsor should make the run better, safer, easier, or more generous.

## Bird&Be-Style Pilates And Fertility Wellness Fund

Use this when the Sponsor sells a sensitive wellness product and needs trust, education, and permission before product trial.

| Component | Setup |
|---|---|
| Campaign | Fertility Wellness And Movement Fund. |
| Event sets | Pilates classes, fertility education circles, nutrition workshops, low-stress movement mornings, creator-led wellness meetups. |
| Host budget | Studio rental, instructor fee, educator fee, tea or light hospitality, take-home kits, printed education, photo only where appropriate. |
| Product role | Optional take-home sample, QR education hub, expert Q&A, partner offer, or post-event follow-up. |
| Proof | Attendance, opt-ins, questions asked, kit claims, qualitative sentiment, Host trust note, and repeat class demand. |

This use case needs extra care:

- make product trial optional
- avoid medical claims unless approved
- let Hosts set privacy norms for photos, attendee lists, and story capture
- prefer education, support, and community over performance marketing language
- use proof that protects attendee dignity

## Product Setup

| Product object | Setup |
|---|---|
| Sponsor | eComm Brand with a product-adjacent ritual. |
| Campaign | Category fund, such as Run Club Performance Fund or Fertility Wellness And Movement Fund. |
| Event sets | Segment by city, Host type, ritual, audience, and proof standard. |
| Hosts | Clubs, instructors, studios, creators, educators, local operators, or venue partners with real audience trust. |
| Host events | Small recurring gatherings with a clear benefit, product role, spend cap, and proof requirement. |
| Spend card | Host stipend, studio or venue fee, hospitality, route support, recovery perk, photo, signage, education materials, or local partner costs. |
| Proof packet | Attendance, claims, content where appropriate, Host note, spend receipts, sentiment, and renewal recommendation. |

## Proof Standard

The first proof packet should answer:

- Did the right community show up?
- Did the Sponsor make the ritual better?
- Did product trial or education feel natural?
- Did the Host want to repeat?
- Did spend stay inside approved rules?
- Which Host, city, or ritual deserves the next allocation?
